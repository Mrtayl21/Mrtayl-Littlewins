How to Use These Resources

  1. Pick one tool to try this week (just one!).
  2. Teach it directly and practice it with students.
  3. Reflect: What worked? What needs tweaking?
  4. Save your improved version for next year.

Tip for New Teachers

Start with routines. When routines run smoothly, everything else feels easier—transitions, behavior, and time management.
